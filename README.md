# 🎮 Classic Games

三款经典小游戏，纯 HTML 单文件实现，浏览器打开即玩。

## 游戏列表

### 🐍 贪吃蛇 · Snake

经典的贪吃蛇游戏，暗色霓虹风格。

- **操作**：↑ ↓ ← → 或 WASD 控制方向，空格暂停
- 蛇身渐变绿色，蛇头带发光效果和眼睛
- 吃食物后速度逐渐加快
- 最高分自动保存到浏览器本地

### 💣 扫雷 · Minesweeper

经典扫雷，三种难度可选。

- **操作**：左键翻开，右键插旗/取消
- **初级** 9×9 / 10 雷 · **中级** 16×16 / 40 雷 · **高级** 16×30 / 99 雷
- 首次点击必定安全，空白格自动展开
- 踩雷揭示全部雷区，标记错误旗子

### 🔗 连连看 · Mahjong Connect

Emoji 版连连看，连接至多 2 个拐弯的折线即可消除。

- **操作**：点击两个相同的 Emoji 消除
- 支持 0 拐（直线）、1 拐（L 形）、2 拐（Z/U 形）路径
- 提示 💡（-10 分）· 洗牌 🔀（-20 分）
- 2 秒内连续消除有 Combo 加分
- 死局自动洗牌
- 键盘快捷键：`H` 提示 · `S` 洗牌 · `R` 重开

## 使用方式

```bash
git clone https://github.com/longwords1/games.git
```

然后用浏览器打开对应的 HTML 文件即可，无需安装任何依赖。
